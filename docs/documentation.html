Manual Penggunaan Plugin

Master Skeleton V4

Ubah desain Figma menjadi kodingan Flutter profesional dalam satu klik.

Kamus Prefix (Naming Rules)

>btn_nama_tombol

Tombol Interaktif

Mengubah layer menjadi GestureDetector. Otomatis menghasilkan boilerplate fungsi onTap dan komentar TODO backend.

>input_nama_field

Formulir Input

Menghasilkan TextFormField lengkap dengan TextEditingController yang terkelola otomatis di State.

>list_nama_daftar

Daftar Dinamis (List)

Mendeteksi pola berulang dan menghasilkan ListView.builder beserta Mock Data Array untuk testing cepat.

>grid_nama_kategori

Tata Letak Grid

Serupa dengan List, namun menghasilkan GridView.builder yang responsif untuk tampilan produk atau galeri.

>appbar_title

Header Aplikasi

Memisahkan layer secara cerdas dan memasukkannya ke properti Scaffold.appBar agar struktur kode sangat bersih.

>bottom_nav_bar

Navigasi Bawah

Menempatkan layer di properti bottomNavigationBar. Tetap terlihat di posisi bawah meskipun halaman di-scroll.

Fitur Cerdas Otomatis

Auto Assets

Semua gambar biner (PNG/JPG) di-export otomatis ke folder assets dan terintegrasi di pubspec.yaml.

Prototype Mapping

Koneksi antar halaman di Figma (Prototype Mode) otomatis dikonversi menjadi Navigator.pushNamed.

Global Theming

Local Styles (Colors) di Figma menjadi class AppColors di Flutter. Ubah di satu tempat, terupdate di semua halaman.

🚀
Pro Tip: Gunakan fitur Auto Layout di Figma. Plugin ini akan membacanya sebagai Row, Column, atau Expanded yang benar-benar responsif!
